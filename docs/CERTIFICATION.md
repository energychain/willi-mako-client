## ğŸ¯ How to Get Certified

### Step 1: Self-Assessment
Fill out the [Certification Self-Assessment Form](https://github.com/energychain/willi-mako-client/discussions/new?category=certification)

### Step 2: Documentation
Provide:
- Production deployment details
- Message volume statistics (anonymized)
- Compliance validation reports
- Use case description

### Step 3: Validation
The Willi-Mako team will:
- Review your submission
- Verify technical implementation (optional code review)
- Validate compliance claims
- Issue certification (typically within 2 weeks)

### Step 4: Maintenance
- Annual re-certification required
- Continuous compliance monitoring
- Community contribution encouraged
